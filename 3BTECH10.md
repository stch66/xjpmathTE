### 教师用书6：人民币的加法

#### 1. **明确这一页的目标和词汇**

**学习目标：**
- 学生应该学会如何进行人民币的加法运算。
- 通过实例，掌握如何将不同面额的钱进行加法计算。
- 理解加法过程中各个部分的组合方式。

**关键词汇：**
- **元**：人民币的基本单位，例如5元、2元。
- **角**：人民币的辅助单位，1元等于10角。
- **加法**：数学运算的一种，将两个或多个数值相加。

**学生可能不理解的词汇：**
- **人民币**：中国的法定货币，包含元、角、分等单位。

#### 2. **数学阅读与写作**

**阅读活动：**
- 通过提问引导学生思考日常生活中的人民币加法应用。例如：你今天买了哪些东西？它们的价格分别是多少？如果你将这些金额加在一起，总共花费了多少？

**数学写作：**
- 请学生用简短的句子描述自己买东西时如何计算价格。例如：“我花了5.3元买了一包奶粉，再花2元买了一束花，最后总共花了7.3元。”

#### 3. **教学呈现与方法**

**课前引导：**
- 通过日常生活中的实际情境来引入本页内容。例如，谷果今天去买奶粉和花，如何计算总共花费了多少钱？
- 通过故事引导学生思考，如何将5.3元和2元相加，得出结果。

**例题展示：**
- 谷果花了5.3元买奶粉，再花2元买花生。请问谷果总共花了多少钱？
  - 解决方法：先将元部分加在一起（5元 + 2元 = 7元），再将角部分加在一起（0.3角 + 0角 = 0.3角）。最后，得出结果：7元 + 0.3元 = 7.3元。

**步骤解析：**
- 第一步：将元部分相加。
- 第二步：将角部分相加。
- 第三步：将结果合并，得出总金额。

#### 3 PB. 教学呈现与方法

**课前引导** ：

- **故事引入** ：讲述一个关于智慧管理财务的圣经故事，如《寡妇的两小钱》（路加福音21:1-4），引导学生思考如何合理分配和计算资源。
- **问题引导** ：提问：“在我们的日常生活中，我们经常需要计算花费的总金额。你们知道如何用数学的方法来解决这些问题吗？”

**例题展示** ：

- CPA方式 

  - **具体（Concrete）** ：准备一些真实的人民币纸币和硬币，让学生通过摆弄这些实物来感受加法运算。
  - **表象（Pictorial）** ：画出模型图，让学生直观地看到不同面额人民币的组合。
  - **抽象（Abstract）** ：列出算式，进行计算。

#### 4. **常见学生错误**

- **错误1**：在加法过程中忽略了角的加法，学生可能直接将5.3元与2元相加为7元，而忽略了0.3角。
  - **纠正建议**：引导学生注意每个单位的加法，确保分开处理元和角的部分。

- **错误2**：混淆了元和角的加法步骤。
  - **纠正建议**：提醒学生先计算元部分，再计算角部分，确保每个单位的加法是清晰独立的。

#### 5. **跨学科内容**

**数学与圣经：**
- 通过圣经中的故事，例如撒玛利亚的好人，来讨论给予和接受的概念，引导学生理解如何分配和计算他们拥有的资源。
- 在计算总费用时，可以用圣经中的分享精神来提醒学生：当我们拥有资源时，我们要善用这些资源。

**数学应用：**
- 在家庭生活中，购买食物、衣物等时，学生常常会遇到加法问题。例如，买菜和水果时如何计算总价。

#### 6. **数学故事**

- **数学故事**：假设有一个小商店，老板小明有5元零钱和3角零钱，他想购买一个价值7.3元的商品。如果顾客给了他5元3角，他是否能收得足够的钱来支付呢？通过这个问题，可以帮助学生了解如何将不同单位的零钱加起来。

#### 7. **笔记**

- 在教学过程中，提醒学生逐步进行计算，先算元，再算角，最后合并结果。
- 提醒学生加法运算中，不同面额的钱需要分开处理，确保不会混淆。

**例题展示：**
- 玛丽花了0.5元买糖果，再花3.2元买沙丁鱼。计算玛丽总共花了多少钱？
  - 解决方法：先将元部分加在一起（3元 + 0元 = 3元），再将角部分加在一起（0.2角 + 0.5角 = 0.7角）。最后，得出结果：3元 + 0.7元 = 3.7元。

#### 7. **笔记**

### 教师用书22：数学应用题——减法与找零

#### **1. 明确这一页的目标和词汇**

**学习目标：**
- 通过实际生活中的应用题，让学生学会如何运用减法来解决实际问题，理解找零的概念。
- 掌握如何从总数中减去已花费的部分，计算出剩余金额，培养学生的逻辑思维和实际操作能力。

**关键词汇：**
- **存款/余额：** 存放的钱或者剩余的钱，是学生们熟悉的概念。
- **花费：** 在购买商品或服务时所支付的金额。
- **找零：** 在支付金额大于商品价格时，卖方给顾客的钱。

#### **2. 数学阅读与写作**

**口头问题：**
- “你有没有买过东西，付完钱后发现自己还剩下多少钱？今天，我们要通过解决一些实际的应用题，学习如何找出剩余的金额！”

**写作题目：**
- “请用简短的句子描述在第一个问题中，如何计算纳吉买雪糕后剩下的钱。通过分解法，你能怎么更容易地解决这个问题？”

#### **3. 教学呈现与方法**

**课前引导：**
- 举例讲解购物时支付和找零的情境。比如，学生们去商店买东西，给了店员35.5元，店员拿去29.3元，那么剩下的钱是多少？
- 提问：“你知道如何从35.5元中减去29.3元吗？让我们一起用减法来计算找零！”

**例题展示：**
- **题目1：** 纳吉有35.5元，买了一盒雪糕后还剩下多少元？
  1. **总数：** 35.5元
  2. **花费：** 29.3元
  3. **剩余金额：** 35.5元 - 29.3元 = 6.2元
  4. **答案：** 纳吉买雪糕后剩下了6.2元。

**教学步骤：**
1. **写出已知条件：** 35.5元和29.3元。
2. **进行减法：** 从35.5元中减去29.3元，得出6.2元。
3. **得出结论：** 学生通过减法计算得出剩余金额。

**题目2：** 谷果的存钱罐里有12元，它花了2.5元买了一杯果汁，再花了7.9元买了一块比萨。请问谷果的存钱罐里还剩多少元？
  1. **总数：** 12元
  2. **总花费：** 2.5元 + 7.9元 = 10.4元
  3. **剩余金额：** 12元 - 10.4元 = 1.6元
  4. **答案：** 谷果的存钱罐里还剩1.6元。

**教学步骤：**
1. **计算总花费：** 2.5元 + 7.9元 = 10.4元。
2. **从总额中减去已花费：** 12元 - 10.4元，得出剩余的1.6元。
3. **得出结论：** 存钱罐里还剩下1.6元。

#### **4. 常见学生错误**

**学生可能的错误：**
- 在进行减法时，学生可能会忘记把小数点对齐，或者误把两个金额的个位、十位数字混淆。
- 学生可能会在解决问题时遗漏步骤，直接跳到答案。

**纠正建议：**
- 强调在进行小数计算时，要对齐小数点，每一位数字的计算都要小心。
- 在解题时，鼓励学生逐步写出每一个计算步骤，确保理解每一部分。

#### **5. 跨学科内容**

  - 另外，圣经中关于“节俭”和“明智使用财富”的教导可以与数学中的理财问题结合起来，帮助学生理解如何在日常生活中运用数学技能，进行有效的财务规划。

#### **6. 数学故事**

### PB 教师用书22：应用题（Qwen 2.5 7B）

#### 1. 明确这一页的目标和词汇
**学习目标**：
- 学生能够通过具体操作、图形表示和抽象计算来解决涉及价格差和剩余金额的问题。
- 学生学会使用减法运算解决实际问题，并能将元换算成角进行更精确的计算。
- 培养学生在日常生活中运用数学知识解决问题的能力，理解数学是神赐予我们认识和管理世界的一种工具。

**关键词汇及定义**：
- **模型图**：一种图形表示方法，用于帮助理解数学问题中的数量关系。例如，在题目中，通过长方形的长度来表示不同物品的价格或金额。
- **价格差**：两个物品或服务之间的价格差异。例如，纳吉买雪糕前后的金额差就是价格差。
- **剩余金额**：在花费一定金额后，剩下的钱数。例如，谷果存钱罐里剩下的钱数。
- **元与角的换算**：1元等于10角。在计算时，需要将元换算成角，以便进行更精确的计算。

**可能的误解**：
- 学生可能会混淆模型图中的各个部分，不清楚每个部分代表什么。
- 在进行元与角的换算时，学生可能会忘记乘以10，导致计算错误。

#### 2. 数学阅读与写作
**口头问题**：
- “纳吉有35.5元，他买了一盒雪糕后还剩29.3元。你能用数学语言描述一下这个问题吗？”
- “谷果的存钱罐里有12元，她花了2.5元买果汁，7.9元买比萨。你能解释一下如何计算她还剩多少钱吗？”

**数学写作题目**：
- “请用一句话描述什么是剩余金额。”
- “请写出你如何使用模型图来解决第一个问题的步骤。”

#### 3. 教学呈现与方法
**课前引导**：
- **故事引入**：讲述一个关于智慧管理财务的圣经故事，如《寡妇的两小钱》（路加福音21:1-4），引导学生思考如何合理分配和计算资源。
- **问题引导**：提问：“在我们的日常生活中，我们经常需要计算花费和剩余的钱。你们知道如何用数学的方法来解决这些问题吗？”

**例题展示**：
- **CPA方式**：
  - **具体（Concrete）**：用实物（如玩具钱）演示价格差和剩余金额的概念。
  - **表象（Pictorial）**：画出模型图，让学生直观地看到价格差和剩余金额。
  - **抽象（Abstract）**：列出算式，进行计算。

**步骤解析**：
##### 第一题：
1. **具体（Concrete）**：准备一些玩具钱，假设每个玩具钱都代表一定的金额。让学生通过摆弄这些玩具来感受价格差。
2. **表象（Pictorial）**：画出模型图，如图所示：
   ```
   纳吉：[-------------------] 35.5元
   雪糕：[------------------] ?
   剩余：[------------------] 29.3元
   ```
   让学生观察并理解纳吉买雪糕前后的金额差。
3. **抽象（Abstract）**：通过模型图，我们可以看到纳吉有35.5元，买雪糕后还剩29.3元。那么，纳吉买雪糕花的钱数就是35.5 - 29.3 = 6.2元。

##### 第二题：
1. **具体（Concrete）**：准备一些硬币，让学生通过数硬币来感受元与角的换算。
2. **表象（Pictorial）**：画出模型图，如图所示：
   ```
   谷果：[-------------------] 12元
   果汁：[------------------] 2.5元
   比萨：[------------------] 7.9元
   剩余：[------------------] ?
   ```
   让学生观察并理解谷果花费和剩余的金额。
3. **抽象（Abstract）**：通过模型图，我们可以看到谷果有12元，她花了2.5元买果汁，7.9元买比萨。那么，谷果总共花了2.5 + 7.9 = 10.4元。所以，谷果的存钱罐里还剩12 - 10.4 = 1.6元。

#### 4. 常见学生错误
- **错误1**：在解题时，学生可能会忘记考虑剩余金额。
  - **纠正建议**：强调在计算花费时要考虑剩余金额，可以通过多次练习来巩固这一概念。
- **错误2**：在进行元与角的换算时，学生可能会忘记乘以10。
  - **纠正建议**：反复练习元与角的换算，确保学生熟练掌握。

#### 5. 跨学科内容
**数学与圣经**：
- 讨论《箴言》3:9-10，理解神赐予我们智慧去管理和分配资源的重要性。

#### 6. 数学故事

  - **讨论**：引导学生思考，为什么神选择用六天来完成创造工作？这与我们日常生活中的数学有什么联系？

#### 7. 笔记
**教学注意事项**：

### 教师用书23：数学应用题——比较与求和

#### **1. 明确这一页的目标和词汇**

**学习目标：**
- 通过实际应用题，引导学生学会用减法比较两个数的大小，并计算它们之间的差异。
- 学会如何将多个数值加起来求总和，帮助学生提高逻辑思维能力和实际操作能力。

**关键词汇：**
- **比较：** 在数学中，比较是指通过减法或其他方式找出两个数之间的差距。
- **总和：** 多个数值加起来的结果。
- **价格：** 商品的价值，可以通过减法或加法来计算差异或总金额。

#### **2. 数学阅读与写作**

**口头问题：**
- “如果你有25.5元，而你的朋友只有18.7元，你能算出你比你的朋友多多少吗？我们将通过减法来解决这个问题！”
- “假设你知道一盒巧克力和一盒饼干的价格，如何计算它们的总价？今天，我们用加法来找出答案。”

**写作题目：**
- “在第一个问题中，皮特有25.5元，苏伊有18.7元。请写下你是如何使用减法来比较这两个数值的差异的。”

#### **3. 教学呈现与方法**

**课前引导：**
- 引导学生通过比较两个数值，理解如何用数学来找出“差额”。例如，比较25.5元和18.7元，计算出你比朋友多了多少钱。
- 提问：“如果我们要计算25.5元和18.7元的差异，应该怎么做呢？”

**例题展示：**
- **题目1：** 皮特有25.5元，苏伊有18.7元。请问皮特比苏伊多多少元？
  1. **已知：** 皮特25.5元，苏伊18.7元。
  2. **计算：** 25.5元 - 18.7元 = 6.8元
  3. **答案：** 皮特比苏伊多6.8元。

**教学步骤：**
1. **列出已知条件：** 25.5元和18.7元。
2. **使用减法：** 25.5元 - 18.7元，得出6.8元。
3. **得出结果：** 皮特比苏伊多6.8元。

**题目2：** 一盒巧克力的售价是24.8元，一盒饼干比一盒巧克力便宜3.4元。请问买一盒巧克力和一盒饼干总共需要多少钱？
  1. **已知：** 一盒巧克力24.8元，一盒饼干便宜3.4元。
  2. **计算：** 饼干的价格 = 24.8元 - 3.4元 = 21.4元
  3. **总价：** 24.8元 + 21.4元 = 46.2元
  4. **答案：** 买一盒巧克力和一盒饼干总共需要46.2元。

**教学步骤：**
1. **首先计算饼干的价格：** 24.8元 - 3.4元 = 21.4元。
2. **计算总价格：** 24.8元 + 21.4元 = 46.2元。
3. **得出结果：** 买一盒巧克力和一盒饼干总共需要46.2元。

#### **4. 常见学生错误**

**学生可能的错误：**
- 在第一个问题中，学生可能会在比较过程中弄错减法的顺序，导致得到错误的差额。
- 在第二个问题中，学生可能会忘记计算饼干的价格，直接将两个数值相加，导致总价错误。

**纠正建议：**
- 强调在进行减法时，必须确保正确的顺序，先将较大的数减去较小的数。
- 在计算总价时，提醒学生先计算每个商品的价格，再加在一起得到总价。

#### **5. 跨学科内容**

- **数学与生活：**
  - 通过这些实际的应用题，学生能学会如何在日常生活中用数学来做预算和比较价格。通过解决“比谁多多少钱”和“总共需要多少钱”这些实际问题，学生能够更好地管理自己的零花钱。

### 教师用书26：开动脑筋（Qwen-vl-72B）

#### 1. 明确这一页的目标和词汇
**学习目标**：
- 学生能够通过具体操作、图形表示和抽象计算来解决涉及价格差的问题。
- 学生学会将元换算成角，并能用数学方法解决实际问题。
- 培养学生在日常生活中运用数学知识解决问题的能力，理解数学是神赐予我们认识和管理世界的一种工具。

**关键词汇及定义**：
- **模型图**：一种图形表示方法，用于帮助理解数学问题中的数量关系。例如，在题目中，通过长方形的长度来表示不同物品的价格。
- **价格差**：两个物品或服务之间的价格差异。例如，番茄比胡萝卜贵2元，这个2元就是价格差。
- **元与角的换算**：1元等于10角。在计算时，需要将元换算成角，以便进行更精确的计算。

**可能的误解**：
- 学生可能会混淆模型图中的各个部分，不清楚每个部分代表什么。
- 在进行元与角的换算时，学生可能会忘记乘以10，导致计算错误。

#### 2. 数学阅读与写作
**口头问题**：
- “米西利亚买了相同数量的番茄和胡萝卜，总共花了18元。你能用数学语言描述一下这个问题吗？”
- “美莲买了一支铅笔和一块橡皮，总共花了1.2元。你能解释一下如何将元换算成角吗？”

**数学写作题目**：
- “请用一句话描述什么是价格差。”
- “请写出你如何使用模型图来解决第一个问题的步骤。”

#### 3. 教学呈现与方法
**课前引导**：
- **故事引入**：讲述一个关于智慧分配资源的圣经故事，如《寡妇的两小钱》（路加福音21:1-4），引导学生思考如何合理分配和计算资源。
- **问题引导**：提问：“在我们的日常生活中，我们经常需要比较和计算不同物品的价格。你们知道如何用数学的方法来解决这些问题吗？”

**例题展示**：
- **CPA方式**：
  - **具体（Concrete）**：用实物（如玩具番茄和胡萝卜）演示价格差的概念。
  - **表象（Pictorial）**：画出模型图，让学生直观地看到价格差。
  - **抽象（Abstract）**：列出算式，进行计算。

**步骤解析**：
##### 第一题：
1. **具体（Concrete）**：准备一些玩具番茄和胡萝卜，假设每个番茄和胡萝卜都代表一定的金额。让学生通过摆弄这些玩具来感受价格差。
2. **表象（Pictorial）**：画出模型图，如图所示：
   ```
   番茄：[-------------------]
   胡萝卜：[------------------] + 2元
   总共：18元
   ```
   让学生观察并理解番茄和胡萝卜之间的价格差。
3. **抽象（Abstract）**：通过模型图，我们可以看到番茄和胡萝卜的总价格是18元，而番茄比胡萝卜贵2元。那么，如果我们将番茄和胡萝卜的价格看作是相等的，那么它们的总价格应该是16元（因为2元被平均分给了两个）。所以，每个胡萝卜的价格是8元，每个番茄的价格是10元。

##### 第二题：
1. **具体（Concrete）**：准备一些硬币，让学生通过数硬币来感受元与角的换算。
2. **表象（Pictorial）**：画出模型图，如图所示：
   ```
   铅笔：[-------------------]
   橡皮：[------------------] + 0.4元
   总共：1.2元
   ```
   让学生观察并理解铅笔和橡皮之间的价格差。
3. **抽象（Abstract）**：通过模型图，我们可以看到铅笔和橡皮的总价格是1.2元，而铅笔比橡皮贵0.4元。那么，如果我们将铅笔和橡皮的价格看作是相等的，那么它们的总价格应该是0.8元（因为0.4元被平均分给了两个）。所以，每块橡皮的价格是0.4元，即4角。

#### 4. 常见学生错误
- **错误1**：在解题时，学生可能会忘记考虑价格差。
  - **纠正建议**：强调在计算总价时要考虑价格差，可以通过多次练习来巩固这一概念。
- **错误2**：在进行元与角的换算时，学生可能会忘记乘以10。
  - **纠正建议**：反复练习元与角的换算，确保学生熟练掌握。

#### 6. 数学故事
**逻辑谜题**：
- **谜题**：有三个人一起去酒店住宿，每人付了10元，共30元。后来老板说今天优惠，只要25元，于是让服务员退还5元。服务员私自藏起了2元，只退还给每人1元。现在每个人实际上付了9元，共27元，加上服务员藏起的2元，总共29元。那么，剩下的1元去哪里了？
